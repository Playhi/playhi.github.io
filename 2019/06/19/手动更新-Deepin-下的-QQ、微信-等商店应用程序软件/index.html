手动更新 Deepin 下的 QQ、微信 等商店应用程序软件

发表于 2019-06-19 更新于 2020-03-23 分类于 教程

Deepin 商店里的 QQ、TIM、微信 这些 wine 的软件，至今版本都有些古老了，不升级能用，但有些新功能就没得用了，今天受论坛帖子启发，成功手动更新了我这 Deepin 上的 QQ 和 微信。

1、在深度商店安装对应软件；
2、打开一次；
3、下载所需软件的更新版本 EXE 到 XX目录；
4、打开深度终端；
5、执行export WINEPREFIX=~/.deepinwine/Deepin-WeChat（参照/home/用户/.deepinwine/下的各个文件夹） //指定位置；
6、执行deepin-wine 文件路径（/XX目录/XX.EXE） //安装；
7、按正常流程安装更新（按钮看不出字的话，鼠标移过去按下（按住并挪开避免执行）左键一般就显示出来了）；
8、安装完毕，检查是否能正常运行，有功能异常的话，接着自己魔改或者卸载重装商店版本。

操作起来吧！BOOM！
